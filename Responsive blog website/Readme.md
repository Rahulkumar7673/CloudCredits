# 📝 Responsive Blog Website
A responsive, modern blog website using HTML, CSS, and JavaScript. Built for clean reading experience across all devices — mobile, tablet, desktop.

# 🔐 Features
📱 Responsive Design — Adapts to mobile, tablet, and desktop screens

📰 Blog Cards Layout — Neatly arranged post previews with images and summaries

🎨 Minimal Dark/Light UI — Clean aesthetics with hover and focus states

🧭 Navigation Bar — Includes links like Home, Articles, About

🧩 Scalable Structure — Easy to expand with more blog posts or pages

💡 Optional Enhancements — Dark mode, animations, blog search bar, etc.

# 📌 Objective
To create a fully responsive, lightweight blog interface using only HTML, CSS, and JavaScript, with emphasis on:

Semantic structure

Mobile-first design

Modular layout for content scalability

Interactive UI features without external libraries

# 🧰 Technologies Used
Technology	Purpose
HTML5	Page structure and semantic elements
CSS3	Styling, responsiveness, hover effects
JavaScript (Vanilla)	Menu toggle, future interactivity (optional)

# ⌨️ Keyboard Input (Optional Interactive Features)
If using JavaScript for keyboard-based navigation or search:

🔎 Search Input: Type to filter blog posts

⌨️ Tab Key: Navigate through blog links

↩️ Enter Key: Open focused blog card or menu item

Esc Key: Close overlays or modals (if implemented)

# ⚙️ How It Works

<footer>: Copyright

Uses Flexbox or CSS Grid for layout

Media Queries for breakpoints (mobile-first)

Transitions for smooth hover animations

🧠 JavaScript (Optional)
If included, JavaScript handles:

Responsive menu toggle (for mobile)

Search/filter functionality

Future enhancements (dark mode toggle, animations)
